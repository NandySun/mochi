## P1 · 加载态：呼吸的圆

**原则：** 动画只在有等待值得标记时出现。本地 SQLite 读盘 < 50ms 不加动画。

| 场景 | 规格 | 触发 |
|---|---|---|
| 网络操作：批量拉取元数据 | 24px 暖米色圆 + 进度文字 | 设置 → 批量拉取全部元数据 |
| 磁盘 I/O：全库扫描 | 16px 圆 + 路径 + 计数，按钮旁内联 | 设置 → 重新扫描 |
| 空库首次启动 | 24px 圆 + 「+ 添加媒体文件夹」 | `series.length === 0` |

**已回退：** 海报懒加载 16px 圆（本地 base64 < 100ms，闪一下就消失）、详情页/播放页加载态（SQLite < 50ms）。

## P2 · 果冻标签 scaleX

在 `filter-indicator`（`layoutId`）之上叠加沿移动方向的 scaleX 变形。

- 正向拉伸 1.08 → 反向 undershoot 0.95 → 归位 1.0，总时长 ~330ms
- `transformOrigin` 设在运动方向反侧，表现「向前探」
- `layoutId` 管位置/尺寸，scaleX 管 transform，两个 CSS 属性互不冲突

## P3 · 按压形变扩展

OSC 栏全面接入 framer-motion 声明式交互，移除所有命令式 `hoverBtn`/`leaveBtn`。

| 控件 | 动画 |
|---|---|
| 播放/暂停 | `whileTap scale=0.88` + `spring.press` |
| 进度条拖拽拇指 | 拖拽时 `scale=1.15`，释放 spring 弹回 |
| 音量滑块拇指 | 同上 |
| 上/下集、倍速、音量图标、轨道、剧集列表 | `whileTap scale=0.92` + `whileHover` 背景色 |

## P4 · 滚动粘度

- 移除 CSS `scroll-snap-type: mandatory` + `scrollSnapAlign: center`
- 用 `useMotionValue` → `useSpring`（stiffness: 100, damping: 35）驱动 `scrollLeft`
- 磁吸：手动滚动停止 120ms 后，距最近卡片 ≤ 30px 时弹簧渐进吸附
- 方向反转：`scrollTarget.set()` 中断前序弹簧，自然产生惯性延迟

**踩坑：** `animate()` 不支持 DOM property `scrollLeft`，改用 MotionValue + 手动写 DOM。

## P5 · 颜色温度叙事

- `src/utils/color.ts`：解析 CSS gradient 字符串 → RGB → HSL → 色相判断冷暖
- `useBackground` 包裹 `setBg`：gradient 变化时自动计算 `tempDirection`
- `GlobalBackground` 梯度层改为 `AnimatePresence` 双层叠加：

| 方向 | 旧层消退 | 新层渗入 |
|---|---|---|
| 冷 → 暖 | 0.9s 缓慢 | 延迟 0.9s 后 0.6s 快速涌入 |
| 暖 → 冷 | 0.45s 快速抽走 | 延迟 0.45s 后 1.05s 缓慢蔓延 |
| 同温 | 0.75s | 同步 0.75s |

## 附修 · PageWrapper 返回按钮位移

**问题：** 详情页返回按钮进入/退出时有肉眼可见的位移。

**根因：** CSS `transform`（含 `translateX(0)` 零值）创建新 containing block，`position: fixed` 元素被迫在父坐标系内定位，随页面 slide 动画一起移动。

**修复：**
1. SeriesDetail 页面动画改为纯 opacity（去掉 `x` 值）
2. PageWrapper 在 enter 动画完成后清除 DOM `transform`，防止残余零值 transform 持续影响 fixed 子元素
