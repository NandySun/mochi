## 路由架构

| 路由 | 组件 | 职责 |
|---|---|---|
| `/` | PosterWall | 海报横滚 + 筛选栏 + 信息 + 观看 |
| `/series/:id` | SeriesDetail | 系列信息 + 剧集列表（纯浏览，无播放） |
| `/play/:episodeId` | VideoPlayer | 纯播放：视频+OSC+侧栏+剧集切换+全屏 |
| `/settings` | Settings | 媒体库/元数据/关于 |

## 全局背景

`GlobalBackground`（App 级三层 fixed 浮层，z:0，pointer-events:none）：
- 各页面通过 `useBackground().setBg()` 控制 gradient/fanart/mask
- 浏览页：完整氛围背景（gradient rgba 0.4 + fanart 0.35 + mask）
- 播放页：gradient 透明 + fanart null，侧栏和剧集区各自带深色底 → 视频透底无雾

## 全屏

- 触发：播放页右上角按钮或 OSC 全屏按钮
- 渲染：`createPortal` 到 `document.body`，`position:fixed; inset:0; background:transparent`
- OSC 3s 无操作自动淡出，鼠标移动重现
- ESC 或点击退出按钮退出全屏
- 视频 margin 全屏时设为 0（填满视口）

## 关键约束

- **mpv 单例**：不销毁重建（避免 native crash）。离开播放页时只 stop 不 destroy
- **refreshTracks 禁用**：`getProperty("track-list","node")` 崩溃（libmpv-wrapper v0.1.1 + libmpv-2 v0.41.0）
- **视频透底**：全屏浮层 `background:transparent`，播放页 gradient 设 `rgba(0,0,0,0)`
- **关闭→隐藏托盘**：`CloseRequested` 拦截 + `window.hide()`
- **useFullscreen 时序**：先 setState 再调 Rust native API，失败回滚
- **resume seek 时机**：必须在 observer 收到 `duration` 事件后执行（文件完全加载），不能在 `loadFile` 返回后立即 seek
- **进度保存**：离开播放页用 `timePosRef.current`（ref 实时值，非 React state）
