# Mochi — 个人多媒体库 设计草案

> 2026-06-16 起草，2026-06-19 讨论敲定，2026-06-20 Phase 1+2 完成。
> 2026-06-21 Phase 3 架构重建 → Beta 里程碑。

---

## 一、要解决的问题

当前本地视频（D:\Video，3 个系列共 30+ 个文件）每次观看需要：
1. 打开文件管理器
2. 逐层导航到目标目录
3. 在扁平文件列表里找到想看的集数

体验等同于「在文件夹里翻文件」，离「浏览影库」差了一个范式。

## 二、核心愿景

一个**桌面原生、封面即导航、多媒介一体的本地影库**：

- 打开应用 → 看到海报墙，每张海报一个系列
- 点击海报 → 进入该系列，看到剧集列表（含封面/缩略图）
- 点击剧集 → 内置播放器直接播放
- 未来扩展漫画、音乐，同一入口管理所有个人媒体
- 跨平台：Windows → macOS 可平滑迁移

## 三、设计原则

1. **桌面原生感**：独立窗口应用，非网页套壳。有托盘、有关联文件类型、有自己的窗口标题栏
2. **海报即导航**：封面网格是唯一入口，不存在侧边栏「文件浏览器」
3. **零配置入库**：指定一个根目录，递归扫描，自动按系列分组
4. **按需拉元数据**：不强制注册 API Key。动漫走 Bangumi（免费无 Key，中文数据），影视走 TMDB（免费 Key），离线优先
5. **本地优先**：所有数据存 SQLite，元数据缓存到本地，断网也能用
6. **渐进式扩展**：v1 只做视频，漫画和音乐是独立里程碑

## 四、技术栈

| 层 | 技术 |
|---|---|
| 框架 | Tauri v2 |
| 前端 | React 19 + TypeScript + Tailwind CSS |
| 数据 | SQLite（rusqlite + bundled） |
| 播放 | libmpv + tauri-plugin-libmpv |
| 元数据 | Bangumi + TMDB 双搜 |

**mpv 嵌入方案**：透明窗口 `transparent: true`，视频帧渲染到原生窗口表面（WebView 之下），WebView 半透明层让视频透出。

## 五、数据模型

```
Series
  ├── title, type(anime/tv/movie/unknown)
  ├── posterPath, fanartPath
  ├── bangumiId / tmdbId
  ├── synopsis, year, score, genres

Episode
  ├── seriesId, seasonNumber, episodeNumber
  ├── filePath, duration, subtitleCount
  ├── status(ready/downloading/missing)
  ├── watchedProgress, watchedCompleted
```

## 六、UI 架构（Beta）

### 路由

| 路由 | 页面 | 职责 |
|---|---|---|
| `/` | PosterWall | 海报横滚 + 筛选栏 + 设置入口 |
| `/series/:id` | SeriesDetail | 系列信息 + 剧集列表，fanart 全屏背景 |
| `/play/:episodeId` | VideoPlayer | 纯播放：视频 + OSC + 信息侧栏 + 全屏 |
| `/settings` | Settings | 媒体库/元数据/关于 |

### 全局背景层

`GlobalBackground`（App 级渲染，三层：gradient + fanart + mask）：
- `position: fixed; z-index: 0; pointer-events: none`
- 跨路由常驻，由各页面通过 `useBackground().setBg()` 控制内容
- 浏览页：完整 gradient(rgba 0.4) + fanart(opacity 0.35) + mask
- 播放页：gradient 透明 + fanart 空，视频区和侧栏各自带深色底

### 首页 `/`

```
┌──────────────────────────────────────────────┐
│  Mochi                              ─  □  ✕  │  标题栏
├──────────────────────────────────────────────┤
│   [fanart 背景 + gradient + mask，全视口]       │  z:0, pointer-events:none
│   全部  动漫  影视                    ⚙ 设置    │  筛选栏
│                                              │
│   ┌──┐  ┌──┐  ┌──┐  ┌──┐  ← 横向滚动         │  海报横滚
│   │  │  │  │  │● │  │  │                    │
│   └──┘  └──┘  └──┘  └──┘                    │
│         标题 · 类型 · ★评分 · 年份              │  信息区
│         "简介..."                     [观看]   │
└──────────────────────────────────────────────┘
```

### 详情页 `/series/:id`

```
┌──────────────────────────────────────────────┐
│  ← 返回                                      │
│                                              │
│   [fanart 背景]                               │
│                                              │
│   ┌──┐  标题                                 │
│   │  │  7.4                                  │
│   │  │  类型 · N集 · ★评分 · 年份              │
│   └──┘  [播放]                                │
│                                              │
│   简介...                                     │
│                                              │
│   剧集                                        │
│   01 02 03 04 05 06 07 08 09                 │
└──────────────────────────────────────────────┘
```

- 海报 hover 出 ▶ 浮层
- 点击海报或 [播放] 按钮 → `/play/:id`（优先续播未看完的）
- 剧集号直接跳转对应播放页

### 播放页 `/play/:episodeId`

```
┌──────────────────────────────────────────────┐
│  ← 返回          E01 · 标题          [ ][ ]   │  顶栏
├──────────────────────────────────────────────┤
│                                              │
│              视频画面                          │  背景透底，mpv 直接透出
│              [OSC 控制条]                      │
│                                              │
├──────────────────────────────────────────────┤
│  01 02 03 04 05 06 07 08 09                 │  剧集切换
└──────────────────────────────────────────────┘
```

- 视频区无雾感：GlobalBackground 播放时退场，非视频区各自带深色底
- 全屏：`createPortal` 浮层注入 `document.body`，OSC 3s 自动隐藏，ESC 退出
- 剧集号点击切换播放（replace 路由，触发重新加载）

### 设置页 `/settings`

独立路由，三区（媒体库/元数据/关于），暗色主题。
